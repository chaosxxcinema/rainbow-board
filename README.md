# rainbow-board
LGBTQ+ Community Mail Visualizer 
The motivation behind this tool is simple: as a member of this marginalized community, my English isn't great. Facing all-English mailing lists always felt overwhelming, and I constantly worried about missing important information.

Since we're all in this together, supporting one another, I wanted to do my part to contribute to our community. So, I wrote this little tool. I hope to use a bit of code to break through the language barrier and information overload, helping friends who share the same struggles, and opening a convenient, safe window for us all.
